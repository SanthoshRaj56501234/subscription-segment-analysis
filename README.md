#  Subscription Segment Analysis

An interactive R Shiny dashboard that analyzes Direct-to-Consumer 
(D2C) subscription segments and tracks revenue trends over time.

##  Business Problem
Subscription businesses need to understand which customer segments 
drive revenue and how segment performance changes over time to 
make smarter retention and pricing decisions.

##  What This App Does
- Visualizes **revenue trends by subscription segment**
- Filters by product and date range interactively
- Identifies returning vs new customer segment behavior
- Built with Plotly for rich interactive charts

##  Technical Stack
R, Shiny, bslib, plotly, tidyverse, lubridate
